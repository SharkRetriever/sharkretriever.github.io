About Me

I am a software developer, a mathematician, and a minimalist.
I am also a fourth-year computer science student at the University of Waterloo.

Software Developer

I’ve been a software developer since high school – I got sick of using jGrasp as the default Java IDE at school, wrote my own in C#, and used it to work on my Java assignments instead.

Nowadays, I work on apps and scripts in a variety of programming languages. For example, the Kotlin Android app UWatALoner helps University of Waterloo students find relatively quiet lecture rooms to study in. There’s also this scraper, a Python script that naively predicts competitor rankings for upcoming Canadian Rubik’s cube competitions.

Mathematician

Proofs are a great way to solidify logical reasoning skills; I’ve written a whole primer on them. Yet, life is often too messy to prove things directly, so I am learning statistics at the moment, hoping to unravel significant insights via data analysis in the future, instead.
