Set theory in organization

Set theory helps people organize their thoughts. Similar ideas can be said to belong in the same “set”. Sets are a bit like boxes, or collections. People buy sets of things. The simplicity of these sentences suggests that sets are not a difficult idea.

Remember that sets can be put inside of sets. This corresponds in organization to writing content with headings and subheadings. In articles of that form, content under a subheading can be a set labelled by the subheading, in which case it is the element of a larger set labelled by its heading. The larger set is itself an element of the article, if we assume an article is a set of sets where each set is identified by its heading.

Set operations are not very applicable to the topic of organization, or self-analysis. However, failing to recognize when one set is a proper subset of another sometimes leads to assuming a one-way implication is a two-way implication, or in other words, a biconditional. This leads to faulty reasoning such as “Jack is a boy, so all boys are Jack”.
