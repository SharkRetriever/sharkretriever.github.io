Face-Turning Octahedron (FTO) Solution


Source: Wikimedia Commons

General Puzzle Notes

The FTO is much more interesting than its corner-turning relative, simply because the corner-turning variant reduces into a hybrid of a pyraminx and an edges-only 3x3x3, which makes it much easier to solve.

Solution found: Feb 26, 2016

General Solving Overview

The puzzle can be solved using just sledgehammer and sune. We define sledgehammer as R’ L R L’ and sune as R U R’ U R U R’ U. Of course, we can also use antisune (R’ U’ R U’ R’ U’ R U’) to help speed things up.

  1. fix corners, using sledgehammer if needed
  2. fix centers using setup moves and sune and/or antisune
  3. fix edges using sune and/or antisune

This is definitely not the most optimal solution, but it is quite nice since there are only two algorithms needed.

To perform these algorithms we must hold the puzzle such that for the U layer, an edge faces you. Thus, it should be possible to do ULRFD, and B is ambiguous. If F is ambiguous or looks more like FL/FR, you’re holding the puzzle wrong. Perhaps do a 1/2-y rotation.

Step 1: Fix Corners

The first layer (bottom layer) corners are very intuitive, especially if one knows how to solve the skewb.

Essentially, any corner can be picked to be the first corner and placed on the bottom layer. A second corner can be placed next to this one, using sledgehammer if necessary. The third corner is then placed as to complete the first layer corners, using sledgehammer if necessary.

To complete the second layer corners, we just need to orient them and AUF.

Find the colour that all second layer corners have in common. This colour will be the colour of the current U side. If only one corner is oriented, turn the puzzle so that the two unoriented corners are in the URF and DFR positions, and the oriented corner is in the FLU position. Then do sledgehammer. Then rotate the puzzle back and AUF.

Step 2: Fix Centers

This step is a bit harder. Any move is a valid setup move, although I recommend having setup moves of at most length 2, since otherwise it becomes harder to use the corners as a reference to help undo them.

What sune does is cycle the U-face centers 90 degrees clockwise. Likewise, antisune cycles the U-face centers 90 degrees counterclockwise.

The very first few centers can probably solved just by performing slice moves, but the centers afterwards will need the sunes and setup moves to complete. Of course, you want to make the setup moves such that after you do the sune and/or antisune, the center is in the proper face (use the corners as references) and that undoing the setup move will not cause the center to just go back to its original position on the wrong face. Thus, the setup move needs to be chosen with some care, and so does the choice of whether the set-up center needs to go clockwise or counterclockwise.

Another tip especially good for bringing the last unsolved center of a side into that wanted (target) side is often first doing a sune or antisune on the target side first so that when the setup move is done, there is clearly a gap that the center can go to after the sune or antisune such that reversing the setup move does not just bring it out of the target side again. This is essentially like keyhole method for the 3x3x3, except we’re using sunes and antisunes to make the keyholes.

Step 3: Fix Edges

What sune does in this step is cycle the U-face edges 90 degrees clockwise. Likewise, antisune cycles the U-face edges 90 degrees counterclockwise.

Note that because the U-face centers are cycled too when using sune/antisune (we use this to our advantage in Step 2), we cannot use setup moves now. This means that we can only use cube rotations, making this step more tedious.

Just like in Step 2, the first few edges can likely be solved directly through one or more applications of sune. Later on though, we might need to use the “keyhole” technique where we purposely perform sune/antisune on a side with two solved edges so that even though that side is now more unsolved, the third unsolved edge can be placed into a solved position relative to the first two edges more easily. Then, undoing the sune (or antisune) setup move completes the face.
