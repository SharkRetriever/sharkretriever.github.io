Largest Comfortable Cube Size and the Decision To Buy A Gigaminx

May 26, 2017

I am considering buying a gigaminx partly out of boredom and partly for the purpose of having something to turn for half an hour straight when there’s nothing else to do, such as when waiting for MiWay buses.

For reference, here is how the gigaminx looks:

But, being the picky cuber I am, I state that I will only buy the gigaminx if it contains fewer cubies than the 9x9x9 Rubik’s Cube. This is because I’m OK with solving a 9x9x9, but anything above really gets tiring. Thus, let’s first calculate the number of cubies in a 9x9x9:

  • There are 8 vertices so we have at least 8 cubies
  • There are 12 edges each with 7 cubies so that gives us 84 more cubies
  • There are 6 centers each with 7*7 = 49 cubies so we add on 294 more cubies

Thus, we have a total of 386 cubies.

Now, let’s calculate the number of cubies in a gigaminx:

  • There are 20 vertices so we have at least 20 cubies
  • There are 30 edges each with 3 cubies so that gives us 90 more cubies
  • There are 12 centers each with 11 cubies so that gives us 132 more cubies

Thus, we have a total of just 242 cubies. Now I actually want a gigaminx.

But, what really interests me more than the gigaminx itself would have to be the following question:

For some integer , what is the smallest xx Rubik’s cube such that the number of cubies it contains is greater than the number of cubies the xx megaminx contains?

The results

  • Megaminx of size 2x2x2 has fewer cubies than a cube of size 3x3x3.
  • Megaminx of size 3x3x3 has fewer cubies than a cube of size 5x5x5.
  • Megaminx of size 4x4x4 has fewer cubies than a cube of size 6x6x6.
  • Megaminx of size 5x5x5 has fewer cubies than a cube of size 8x8x8.
  • Megaminx of size 6x6x6 has fewer cubies than a cube of size 9x9x9.
  • Megaminx of size 7x7x7 has fewer cubies than a cube of size 11x11x11.
  • Megaminx of size 8x8x8 has fewer cubies than a cube of size 13x13x13.
  • Megaminx of size 9x9x9 has fewer cubies than a cube of size 14x14x14.
  • Megaminx of size 10x10x10 has fewer cubies than a cube of size 16x16x16.
  • Megaminx of size 11x11x11 has fewer cubies than a cube of size 17x17x17.
  • Megaminx of size 12x12x12 has fewer cubies than a cube of size 19x19x19.
  • Megaminx of size 13x13x13 has fewer cubies than a cube of size 20x20x20.
  • Megaminx of size 14x14x14 has fewer cubies than a cube of size 22x22x22.
  • Megaminx of size 15x15x15 has fewer cubies than a cube of size 24x24x24.

Conclusion

So, not only does the gigaminx have fewer pieces than the 9x9x9 Rubik’s cube, but the 6x6x6 minx also has fewer pieces than the 9x9x9 Rubik’s cube. Doesn’t mean I’ll buy the latter, though.
